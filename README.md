# n8n-nodes-passwork

This is an n8n community node. It lets you use self-hosted _Passwork_ in your n8n workflows.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.
